Detailed Description

Implementation of a single aDDMTrial object.

An aDDMTrial can either be generated by simulating data trials or loading from a CSV. A single trial represents an individual binary perceptual choice made by one subject. These trials can be aggregated together for model fitting and likelihood computations.

fixItemVector of integers representing the items fixated on during the trial in chronological order. 1 corresponds to left, 2 corresponds to right, and any other value is considered a transition or blank fixation.
fixTimeVector of integers corresponding to the duration of each fixation. Must be equal in size to fixItem.
fixRDVVector of floats corresopnding to the RDV values at the end of each fixation.
uninterruptedLastFixTimeInteger corresponding to the duration (milliseconds) that the last fixation in the trial would have if it had not been terminated when a decision had been made.
