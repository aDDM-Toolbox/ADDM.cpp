Detailed Description

Information pertaining to the computation of likelihoods for a dataset of trials (either DDM or aDDM).

The ProbabilityData class provides a wrapper for information that may be returned after computing the likelihoods for a set of trials. The class is meant to provide output information that can be used when returning NLLs or marginalized posteriors in MLE.
