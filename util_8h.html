Create empirical distributions fro the data ot be used when generating model simulations.

Parameters
dataA mapping of subject IDs to vectors of each subject's corresponding aDDMTrials.
timeStepMinimum duration of a fixation to be considered in milliseconds.
maxFixTimeMaximum duration of a fixation to be considered, in milliseconds.
numFixDistsInteger indicating the number of fixation types to use in the fixation distributions. I.e. if set to 3, then three separate fixation typyes will be used, corresponding to the first, second, and third fixation in each trial.
valueDiffsList of integers corresponding to the available value differences between item.
subjectIDsList of subject IDs to consider in the empirical data. If left empty, all subjectIDs will be used.
useOddTrialsBoolean indicating whether or not to use odd trials when creating the distributions.
useEvenTrialsBoolean indicating whether or not to use even trials when creating the distributions.
useCisTrialsBoolean indiciating whether or not to use cis trials when creating the distributions. Cis trials are defined as trials in which both items have either positive or negative values.
useTransTrialsBoolean indicating whether or not to use trans trials when creating the distributions. Trans trials are defined as trials in which one item has a positive value and one item has a negative value.
Returns
FixationData object serving as a record of empirical fixation distributions.
